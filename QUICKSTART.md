# 🚀 快速启动指南

5分钟让你的明星个人站运行起来！

## ⚡ 三步启动

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动开发服务器
```bash
npm run dev
```

### 3️⃣ 访问网站
打开浏览器访问：**http://localhost:3000**

🎉 就这么简单！

## 📸 添加图片（可选）

网站已经可以运行，但你可能想添加真实图片：

### 方式1: 使用占位图（推荐快速测试）
网站会自动使用占位图路径，虽然图片不会显示，但功能完全正常。

### 方式2: 添加真实图片
```bash
# 将你的图片复制到对应目录
public/images/
  ├── banner/          # 首页轮播图
  ├── works/           # 作品海报
  ├── highlights/      # 高光时刻
  ├── brands/          # 品牌代言
  ├── magazines/       # 杂志封面
  └── gallery/         # 图库
```

详细的图片规格请查看 `public/images/README.md`

保存后，页面会自动刷新！

## 🎨 更换主题色

编辑 `src/app/globals.css`：

```css
:root {
  --purple-600: #9333ea;  /* 改成你喜欢的颜色 */
}
```

## 🆘 遇到问题？

### 安装失败
```bash
# 清除缓存重试
rm -rf node_modules package-lock.json
npm install
```

### 端口被占用
```bash
# 使用其他端口
npm run dev -- -p 3001
```

### 页面空白
1. 检查浏览器控制台是否有错误
2. 确认开发服务器正在运行
3. 尝试硬刷新：Ctrl/Cmd + Shift + R
